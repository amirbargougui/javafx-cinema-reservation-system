# 🎬 Cinema Reservation System (JavaFX Desktop Application)

A full-stack desktop application for managing cinema reservations, built using JavaFX and MVC architecture.  
The system allows users to browse films, reserve seats, manage bookings, and provides an admin dashboard for management operations.

---

## 🚀 Features

### 👤 User Side
- Browse available movies and cinema sessions
- View film details and schedules
- Reserve seats for a selected session
- Generate reservation confirmation (PDF / QR code support)
- Manage user profile

### 🛠️ Admin Side
- Add / edit / delete films
- Manage categories and cinema sessions
- Manage users and reservations
- View statistics and reports
- Control system data (CRUD operations)

---

## 🧱 Architecture

The project follows the **MVC (Model - View - Controller)** pattern:

- **Models** → Business entities (Films, Users, Reservations, etc.)
- **Controllers** → Application logic and UI handling
- **Services** → Business logic and database interaction
- **Views** → JavaFX UI (FXML-based interfaces)

---

## 🛠️ Tech Stack

- Java 8+
- JavaFX (Desktop UI)
- FXML (UI design)
- MySQL (Database)
- Maven (Build tool)
- JDBC (Database connection)
